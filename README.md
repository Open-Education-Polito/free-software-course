# Free Software Course Polito Website
This is the official repository of the website for the *Free Software Course*
at Politecnico di Torino. 

## Warning
In order to be used, a repo structure has to exist. See this folder structure:
```
README.md                                                                 
repo                                                                      
├── CATEGORIES
----- PDF
----- SRC
```

Categories name can be consistent with the name of the `PHP` files present
here since the variable `$category` contains the name of the folders.
